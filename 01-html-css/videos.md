# Videos – Bloque 01: HTML y CSS

Estos son recursos recomendados para aprender y repasar HTML y CSS.  
No tienes que verlos todos de golpe; puedes ir combinando según tu ritmo.

---

## 1. Learn HTML – Full Tutorial for Beginners (freeCodeCamp)

**Link:** https://www.youtube.com/watch?v=kUMe1FH4CHE  

Curso completo de HTML para principiantes. Ideal para entender bien la estructura de un documento HTML5 y las etiquetas principales.

---

## 2. Learn HTML & CSS – Full Course for Beginners (freeCodeCamp)

**Link:** https://www.youtube.com/watch?v=a_iQb1lnAEQ  

Curso donde aprendes HTML y CSS construyendo varios proyectos. Muy útil para ver cómo se combinan ambas cosas en páginas reales.

---

## 3. HTML Crash Course For Absolute Beginners (Traversy Media)

**Link:** https://www.youtube.com/watch?v=UB1O30fR-EE  

Crash course de HTML: en un solo video te lleva desde cero hasta tener una base sólida para empezar a experimentar.

---

## 4. HTML & CSS for Absolute Beginners (Kevin Powell – Playlist)

**Link:** https://www.youtube.com/playlist?list=PL4-IK0AVhVjOJs_UjdQeyEZ_cmEV3uJvx  

Playlist pensada para principiantes. Explica HTML y CSS con mucho foco en entender el *por qué* de las cosas, no solo copiar código.

---

## 5. Curso HTML para Principiantes (Fazt – Español)

**Link:** https://www.youtube.com/watch?v=rbuYtrNUxg4  

Curso en español de HTML para quienes prefieran repasar conceptos en su idioma. El mismo canal tiene curso de CSS que complementa muy bien.
