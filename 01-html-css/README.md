# Bloque 01 – HTML y CSS

En este bloque vamos a construir nuestra **primera página web personal** usando solo **HTML** y **CSS**.  
La idea es que tengas algo parecido a un **CV online**, pero con una interfaz más visual y profesional.

## Objetivos

- Entender la estructura básica de un documento HTML5.
- Usar etiquetas semánticas: `<header>`, `<main>`, `<section>`, `<footer>`, etc.
- Aplicar estilos con un archivo CSS externo (`style.css`).
- Diseñar una página personal tipo CV que se vea limpia y profesional.
