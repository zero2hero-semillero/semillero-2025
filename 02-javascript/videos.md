# Videos - Bloque 02: JavaScript

Recursos recomendados para aprender JavaScript moderno y aplicarlo a tu CV interactivo. No tienes que verlos todos de una: avanza segun tu ritmo y practica con tu propio codigo mientras miras los ejemplos.

---

## 1. JavaScript Programming - Full Course (freeCodeCamp)

**Link:** https://www.youtube.com/watch?v=jS4aFq5-91M  
Curso completo en ingles. Cubre sintaxis base, funciones, objetos, arreglos, DOM y eventos. Ideal para entender el lenguaje sin depender de frameworks.

---

## 2. JavaScript DOM Crash Course (Traversy Media)

**Link:** https://www.youtube.com/watch?v=0ik6X4DJKCc  
Enfocado en manipular el DOM: seleccionar elementos, cambiar estilos, crear nodos y escuchar eventos. Perfecto para dar vida a tu pagina personal.

---

## 3. Asynchronous JavaScript Tutorial - Promises y async/await (Web Dev Simplified)

**Link:** https://www.youtube.com/watch?v=PoRJizFvM7s  
Explica la base de asincronia y como trabajar con datos externos o temporizadores sin bloquear la interfaz.

---

## 4. 15 Vanilla JavaScript Projects (freeCodeCamp)

**Link:** https://www.youtube.com/watch?v=3PHXvlpOkf4  
Coleccion de mini proyectos sin frameworks. Te ayuda a ver patrones comunes de interactividad (carouseles, contadores, modales, filtros, etc.).

---

## 5. Curso de JavaScript desde cero (midudev - Espanol)

**Link:** https://www.youtube.com/watch?v=z95mZVUcJ-E  
Curso en espanol que repasa bases, buen estilo de codigo y ejemplos practicos para trabajar en el navegador.
