# Bloque 02 - JavaScript

En este bloque vas a tomar la pagina personal del Bloque 01 y darle dinamismo con **JavaScript**. El objetivo es que tu CV web ya no sea solo estatico: deberia responder a la interaccion de la persona que lo visita, mostrar datos generados por codigo y ofrecer una experiencia mas viva.

## Objetivos

- Entender como manipular el DOM para leer y actualizar contenido.
- Escuchar eventos (click, submit, input) y reaccionar sin recargar la pagina.
- Manejar datos en arreglos/objetos para renderizar listas y filtros.
- Separar responsabilidades en archivos: `index.html`, `style.css` y `script.js`.
- Aplicar buenas practicas basicas: nombres claros, funciones pequenas, evitar codigo duplicado.

## Entregable de este bloque

- En tu repositorio personal:
  - Carpeta: `02-javascript/reto-02-interactividad-cv/`
  - Archivos dentro:
    - `index.html` (puedes partir del de Bloque 01 y mejorarlo).
    - `style.css` (ajustes de estilos si los necesitas).
    - `script.js` (toda la logica y eventos van aqui, no en linea).

El mentor revisara que el sitio siga viendose bien, que la interaccion funcione sin errores en consola y que el codigo JavaScript sea legible y bien organizado.
