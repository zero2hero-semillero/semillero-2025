# Bloque 03 - Bootstrap

En este bloque vas a construir una **landing page** para un estudio de tattoos usando **solo Bootstrap**. El foco es aprender a maquetar rapido, mantener consistencia visual y aprovechar componentes listos sin escribir CSS o JavaScript propios.

## Objetivos

- Usar el sistema de grid de Bootstrap para layouts responsive.
- Aplicar componentes listos: navbar, cards, carousel, accordion, forms, badges, etc.
- Dominar utilidades de espaciado, tipografia, colores y alineacion.
- Integrar Bootstrap via CDN y activar componentes con `data-bs-*`.
- Mantener un HTML limpio y ordenado, sin estilos ni scripts personalizados.
